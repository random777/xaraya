7.13. Package Your Module

At this stage the module should be ready for packaging. The two most widely used packaging formats are WinZip (.zip extension) and compressed TAR (.tar.gz extension). If possible, package the module with both formats. If not then just package it with the format that you have and ask someone on the Xaraya modules site if they can package it in the other format.
