Xaraya Module Developers Guide

  •     What is a module?

The Xaraya system allows for expansion of its functionality using modules. A module is a set of files containing functions with pre-defined names and roles that integrate very easily with a standard deployment of Xaraya. A module can also include blocks, images, plain HTML files etc.
