Chapter 4. Xaraya Module Operations

This chapter covers how modules interact with Xaraya. The information in this chapter is correct for the 0.71 release of Xaraya, for other releases please get the most recent copy of the Module Developers Guide.

4.1. Locating Modules

All Xaraya modules must be placed within their own subdirectory of the 'modules' directory to be recognised. Modules placed anywhere else within the filesystem will not be located correctly.
