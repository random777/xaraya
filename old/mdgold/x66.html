1.7. Suggestions and Updates

The Xaraya module system is a work-in-progress. There are no doubt many good ideas out there that have not been incorporated into the Xaraya module system, and if a developer has a request for a particular set of functionality then they can submit it to the Xaraya features request list at the Xaraya Homepage. If you have found a bug within the current module system then you can submit it to the bug list at the same address.

Please note that the main requirement for the Xaraya module design is stability. Due to this it is possible that your request for new or updated functionality will get refused on the grounds that it is too specific, can easily be built from core API functions, or carries out work that should rightly be done by a module. In such situations the Xaraya team will always try to provide a simple alternative, but please remember that submission of a new or updated addition to the module design does not guarantee inclusion.
