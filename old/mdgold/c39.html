Chapter 1. Introduction

1.1. What is a Module?

The Xaraya system allows for expansion of its functionality through the use of modules. A module is set of files containing functions with pre-defined names and roles that integrate very easily with a standard deployment of Xaraya. A module can also include blocks, images, plain HTML files, etc.
