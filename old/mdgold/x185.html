3.3. Single Directory Installation

Having a single install directory allows for much easier maintenance of large Xaraya systems, and far easier install and removal of modules both for the module developer and for the site administrator. Dependencies of the layout on the filesystem are no longer required, and as such the module designer does not need to worry about on which systems his module might be deployed, and how it needs to interact with the underlying operating system to function correctly.
