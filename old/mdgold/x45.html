1.2. Why Write a Module?

There are a number of reasons to write a module. The main reason is because Xaraya does not provide a specific function that you would like it to. Examples of modules that have been developed to date for Xaraya include bulletin boards, galleries, calendars, address books, and MP3 search utilities.
